# backdoor-http

Simple implementation in nodejs of http backdoor for revshell

## Start payload (target)

```bash
npm run payload
```

## Start C2 (client)

```bash
npm run c2
```

## Example

- Target:

```bash
git clone https://github.com/miguelzacca/backdoor-http.git
cd backdoor-http
npm run payload
```

- Client:

```bash
git clone https://github.com/miguelzacca/backdoor-http.git
cd backdoor-http
npm run c2
```
